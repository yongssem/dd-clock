# 뚝딱 시계 (dd-clock)

교실 스크린용 픽셀 아트 스타일 시계.

- **큰 시계** · 18rem 시/분/초
- **날짜 + 요일**
- **현재 교시** 자동 표시 (수업 중 / 쉬는 시간 / 등교 / 방과 후)
- **3가지 테마** · 밝은 / 어두운 / 파스텔
- **교시 시간표 편집** · ⚙️ 버튼 → localStorage 저장

## 실행
`index.html`을 브라우저로 열면 끝. 빌드 없음.

## 배포
Vercel에 Framework Preset **Other**로 추가.

## 기술 스택
- 순수 HTML/CSS/JS (단일 파일)
- Galmuri11 픽셀 한글 폰트 (CDN)
